# 💬 Chat App Frontend

Real-time Chat Application Frontend built with React.js & Socket.IO.

## 🚀 Features
- Real-time Messaging (Socket.IO)
- User Authentication (JWT)
- Group Chat & Private Chat
- Online/Offline Status
- Emoji Support
- Responsive Design

## 🛠️ Tech Stack
**Frontend:** React.js, Redux, Tailwind CSS

**Real-time:** Socket.IO

**Other:** JWT, Cloudinary

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/poonammgambhire/Chat_App_Frontend.git

# Install dependencies
npm install

# Run
npm run dev
```
